Bank Transactions Management Application
This is a bank transactions management application built with React, Redux Toolkit, and Vite for the frontend. The application allows users to view and manage their bank transactions.

Features
User authentication with JWT tokens
View all transactions for the current month, grouped by account
View details of a specific transaction
Edit transaction information
Clear specific fields of a transaction
Technologies Used
Frontend: React, Redux Toolkit, Vite
Installation and Setup
Prerequisites
Ensure you have the following installed on your machine:

Node.js (>=14.x)
npm (>=6.x)
Cloning the Repository
bash
Copier le code
git clone https://github.com/yourusername/bank-transactions-app.git
cd bank-transactions-app
Install Dependencies
For the Frontend
bash
cd frontend
npm install
For the Backend
bash
cd backend
npm install
Running the Application
Start the Frontend
Navigate to the frontend directory and run:

bash
npm run dev
This will start the Vite development server. You can access the application at http://localhost:5173/.

Start the Backend
Navigate to the backend directory and run:

bash
npm run dev:server
This will start the Express server. The backend API will be accessible at http://localhost:3001.
